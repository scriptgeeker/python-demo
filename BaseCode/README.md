## 字节码编码

模拟从二进制到字符的编码过程，封装Base64编解码工具

### 核心原理

Base64是一种用64个字符来表示任意二进制数据的方法。

字节数据转换过程

Base64编码会把3字节的二进制数据编码为4字节的文本数据。
